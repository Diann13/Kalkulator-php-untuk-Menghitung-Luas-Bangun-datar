# 📖 Tentang Project

**PHP Geometry Area Calculator** merupakan aplikasi web sederhana berbasis PHP yang digunakan untuk menghitung **luas beberapa bangun datar** berdasarkan data yang dimasukkan oleh pengguna.

Project ini saya kerjakan ketika masih menempuh pendidikan di **SMK Negeri 1 Pacitan** jurusan **Rekayasa Perangkat Lunak (RPL)** sebagai latihan dasar pemrograman web menggunakan **PHP** dan **HTML**.

Repository ini saya simpan sebagai dokumentasi perjalanan belajar saya dalam memahami dasar-dasar pengembangan aplikasi web.

---

# ✨ Fitur

Program menyediakan perhitungan luas untuk beberapa bangun datar, yaitu:

* ⬜ Bujur Sangkar
* ▭ Persegi Panjang
* ⭕ Lingkaran
* 🔺 Segitiga

Setiap bangun datar memiliki form input yang berbeda sesuai kebutuhan perhitungannya.

---

# 🧮 Rumus yang Digunakan

| Bangun Datar    | Rumus             |
| --------------- | ----------------- |
| Bujur Sangkar   | Sisi × Sisi       |
| Persegi Panjang | Panjang × Lebar   |
| Lingkaran       | π × r²            |
| Segitiga        | ½ × Alas × Tinggi |

Pada project ini nilai π menggunakan **3.14**.

# 🔄 Alur Program

Program berjalan melalui tiga halaman:

```text
index.php
      │
      ▼
Memilih Bangun Datar
      │
      ▼
hitung.php
      │
Input Nilai
      │
      ▼
hasil.php
      │
Menampilkan Luas Bangun Datar
```

# ▶️ Cara Menjalankan

### 1. Clone Repository

```bash
git clone https://github.com/Diann13/Kalkulator-php-untuk-Menghitung-Luas-Bangun-datar.git
```

### 2. Simpan Project

Pindahkan folder project ke:

```text
htdocs/
```

jika menggunakan XAMPP.

---

### 3. Jalankan Apache

Aktifkan Apache melalui XAMPP Control Panel.

---

### 4. Akses Melalui Browser

```text
http://localhost/Kalkulator-php-untuk-Menghitung-Luas-Bangun-datar
```

---

# 📖 Cara Menggunakan

1. Pilih salah satu bangun datar.
2. Tekan tombol **Hitung**.
3. Masukkan nilai yang diperlukan.
4. Tekan tombol **Hitung** kembali.
5. Program akan menampilkan hasil luas bangun datar.
